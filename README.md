# react-iron-boilerplate
React.js + Iron framework

## Features

### Client
* ES2015 + Babel + Webpack
* React + Semantic UI React

### Server
* Rust (stable channel)
* iron (HTTP framework)
* Tera (template engine)

## Usage
```sh
$ npm run compile
$ cargo run
```

## Configuration

```sh
# .env

BIND_ADDRESS=0.0.0.0:3000
# ...
```
